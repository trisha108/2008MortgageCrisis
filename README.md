## Project Overview  
This project analyzes how the 2008 financial crisis reshaped U.S. mortgage lending using HMDA data from 2007–2017. It focuses on uncovering macro trends, shifts in lending behavior, and regional recovery patterns through interactive and narrative-driven visualizations.

## Dataset  
- HMDA Historic Data (2007–2017)  
  https://www.consumerfinance.gov/data-research/hmda/historic-data/

## Tools & Technologies  
- Python (Google Colab) – Data cleaning, preprocessing, and aggregation  
- Tableau – Data visualization and dashboard development  

## Key Insights  

### Geographic Exploration  
- Identified regional disparities in mortgage activity  
- Highlighted which areas were most affected during the crisis  
- Tracked which regions recovered fastest over time  

### Compositional Shifts  
- Analyzed changes in loan purpose (purchase vs. refinance)  
- Examined shifts between conventional and government-backed loans  
- Revealed evolving borrower behavior and lender risk appetite  

### Recovery Trends  
- Tracked the decline and rebound of mortgage applications and originations  
- Identified the lowest point of the market crash  
- Pinpointed early signals of recovery across segments  

## Deliverables  
- Presentation deck with key visualizations and insights  
- Tableau dashboards for interactive exploration
